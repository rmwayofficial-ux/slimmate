# 🌿 slimmate — スリムの相棒

運動と食事を記録して、ゆるキャラ「あるるん」と毎日つづける PWA。スタンプ・称号レベル・着せ替えで楽しく習慣化できます。

**公開URL:** https://rmwayofficial-ux.github.io/slimmate/

![slimmate](og-image.png)

## できること

### 📝 基本の記録
- 🏃 **運動記録** — 種別×時間で消費カロリーを自動計算（METs × 体重 × 時間 × 1.05）
- 🍽️ **食事記録** — 写真添付＋カロリープリセット、摂取/消費バランス表示
- 📅 **過去日付に記録** — 日付バーで前の日に切り替えて遡って入力
- ✏️ **記録の編集** — 時間・カロリー・メモの後修正に対応
- ⭐ **お気に入り運動・食事** — チップ右上の☆で先頭にピン留め
- 🏷️ **カスタム種目追加** — オリジナルの運動（METs指定）・食事（kcal指定）

### 💪 健康習慣を見える化
- 💧 **水分記録** — 1日8杯のコップカウンタ
- ⚖️ **体重トラッキング** — SVG折れ線グラフで推移と変化量を表示
- 🍪 **間食メモ** — ワンタップで「がまんした✊」「食べちゃった🍪」を記録

### 🎁 モチベ・ご褒美
- 🏆 **トロフィースタンプ＋連続記録** — 達成日に紙吹雪付きトロフィー
- 🏅 **称号レベル** — たまご🥚 → ルーキー🌱 → ブロンズ🥉 → シルバー🥈 → ゴールド🥇 → プラチナ🏆 → ダイヤ💎
- 🎀 **マスコット着せ替え** — リボン/キャップ/おうかん/サンタ（達成日数でアンロック・全無料）
- 🔮 **今日のおみくじ** — 毎日のおすすめ運動を1つ提案、引き直し可
- 🎁 **達成カード自動生成** — Canvas で1080×1080 PNG生成、SNSシェア対応
- 📊 **振り返りグラフ** — 週間/月間カロリー（消費 vs 摂取）
- 📜 **月別総まとめ** — 達成日数・総消費・一番やった運動などの成績表

### 🔔 声・通知
- 🔔 **あるるんアラーム** — 指定時刻から一定間隔で「みてるよ」と鳴り続ける（停止まで継続）
- ⏰ **モーニング/ナイトコール** — 朝晩の声かけ＋通知
- 🍪 **間食ナッジ** — 「がまんした→えらいぞー」「食べちゃった→のーーー」を声で反応
- 🔊 **声の高さ・速さ調整** — Web Audio オルゴール＋speechSynthesis

### 🌙 見た目・写真
- 🌙 **ダークモード** — auto / light / dark（OS連動）
- 🖼️ **食事写真ギャラリー** — これまでの食事写真をグリッド表示
- 📱 **iOSホーム画面追加ガイド** — 初回バナーで案内

### 💌 ヘルプ・フィードバック
- ❓ **使い方ガイド・FAQ** — アプリ内ヘルプ（ヘッダーピル/ホームCTA/設定CTAの3カ所から開ける）
- 💌 **ご意見送信** — Supabase に直接送信（カテゴリ・星評価・本文）

### 💾 データ
- 端末内 localStorage のみで完結。ログイン不要・記録データはサーバーに送信しません
- 📄 テキスト書き出し（人間が読む用）
- 💾 JSONバックアップ／📥復元（機種変対応）

## 技術

- 依存ゼロのバニラ JavaScript PWA（ビルド不要）
- 記録データは端末内（localStorage）に保存。サーバー送信なし・ログイン不要
- Service Worker でオフライン対応・ホーム画面追加対応（network-first）
- フィードバック送信のみ Supabase REST API を使用（RLS で INSERT のみ許可）

## ローカルで動かす

```bash
python3 -m http.server 8000
# → http://localhost:8000/
```
