# 🌿 slimmate — スリムの相棒

運動を記録して消費カロリーを見える化。ゆるキャラ「あるるん」とトロフィースタンプで毎日続く、あなたのスリム相棒アプリです。

**公開URL:** https://rmwayofficial-ux.github.io/slimmate/

## できること

- 🏃 **運動記録** — 種別×時間で消費カロリーを自動計算（メモ欄つき）
- 🍽️ **食事記録** — 写真添付＋カロリー目安、摂取/消費バランス表示
- 🏆 **スタンプ＆連続記録** — 達成日に華やかなトロフィー、ストリーク表示
- 📊 **振り返り** — 週間／月間のカロリーグラフ
- 🔔 **あるるんアラーム** — 設定時刻から一定間隔で「みてるよ」と声かけ（停止するまで）
- 🍪 **間食ナッジ** — 「がまんした→えらいぞー」「食べちゃった→もーーー」と声で反応
- 💾 **バックアップ／復元** — 端末内データの書き出し・読み込み

## 技術

- 依存ゼロのバニラ JavaScript PWA（ビルド不要）
- データは端末内（localStorage）に保存。サーバー送信なし・ログイン不要
- Service Worker でオフライン対応・ホーム画面追加に対応

## ローカルで動かす

```bash
python3 -m http.server 8000
# → http://localhost:8000/
```
